# Retail Sales Analysis (Excel & SQL)

## Objective
Analyze retail sales data to identify revenue trends, top products, and regional performance.

## Tools Used
- Excel (data cleaning, pivot tables, dashboards)
- MySQL (data querying)
- GitHub (project hosting)

## Project Structure
- data/: raw and cleaned Excel files
- sql/: SQL queries
- images/: analysis screenshots

## Key Insights
- Rice generated the highest revenue
- The West region performed best
- Sales peaked in February
